>>> 4.  How memory is managed in Python? 

>>> In Python, memory management is handled by the Python memory manager, which is responsible for allocating and deallocating memory for objects in the Python program. Python uses a private heap to manage memory, and it has its own built-in memory allocation and deallocation mechanisms.
